## 개요
> 갈보리교회 초등부 출석부 DB화를 위한 프로젝트
> > 기능정의 - 기능 써주세요
> > * 기능 


## 서버 사용방법
~~~
서버상에서 실행해보기
cd calvary
npm i
npm run dev 
~~~

pm2를 사용한 백그라운드 실행
(터미널을 종료해도 서버유지)
pm2 npm -- run dev
* * * 

## 현재 기능
매주 출석 및 미션 달란트 집계 기능
특정 월 또는 전체 생일자 조회

## 추가 기능
학생 반, 정보 등 편집 기능
학생의 반 히스토리 기능
